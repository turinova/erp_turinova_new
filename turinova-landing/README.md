# Turinova Landing Page

Professional landing page for Turinova ERP.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server (port 3003)
npm run dev

# Build for production
npm run build
npm start
```

## URLs

- **Development:** http://localhost:3003
- **Production:** https://turinova.hu

## Deploy to Vercel

```bash
vercel --prod
```

Then configure domain `turinova.hu` in Vercel dashboard.
